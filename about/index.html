About the app

Designed to make every chain reaction readable.

Chain Reaction MAX keeps the classic chain-reaction formula, with a cleaner Flutter interface, flexible board presets, and replay-friendly feedback.

Why it exists

The goal is simple: keep the gameplay chaotic, but make each decision easy to read. Setup is quick so matches start fast without menu friction.

What already ships

The current build includes turn logic, elimination handling, explosion replay, and session setup flow. This site documents what is already working in the app.

How it should feel

Match readability matters more than visual noise. The UI is intentionally compact so players can focus on timing, pressure, and board control.

Core pillars

01

Every move changes the map

Corners pop at two atoms, edges at three, and center cells at four. That shifting critical mass keeps the board readable and tense.

02

Play around one screen

Spin up a game instantly and pass the device between players.

03

Built for dramatic reversals

Captured cells immediately flip ownership during a blast, so one perfectly timed burst can erase a lead and change the whole table mood.

04

Replay the last explosion

The game records every explosion step and can replay the previous chain sequence so players can study exactly how the board swung.

05

Undo the last move

A full move history is kept throughout each game, so any player can undo the most recent atom placement while a match is still live.

06

Sessions persist and resume

Active games are saved automatically after every turn. Any unfinished match can be picked up from the home screen without losing board state or turn order.
